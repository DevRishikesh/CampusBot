# 🎓 CampusBot – WhatsApp Bot for Schools

CampusBot is a smart WhatsApp automation system designed for schools and colleges to streamline communication with parents and students.

It acts as a **24/7 virtual assistant**, instantly responding to queries, sending reminders, and reducing manual workload for school staff.

---

## 🚀 Features

* 📲 **24/7 Auto Replies** – Instantly answer common parent queries
* 💰 **Fee Reminders** – Automatically notify parents before due dates
* 📅 **Exam Timetable Info** – Share schedules instantly
* 📢 **Broadcast Messages** – Send announcements to all parents
* 🤖 **AI Smart Replies** – Handle unknown questions naturally
* 🌐 **Multi-language Support** – Communicate in regional languages
* ⚡ **No App Required** – Works directly on WhatsApp

---

## 🎯 Problem It Solves

Schools often face:

* Missed calls from parents
* Repetitive queries handled manually
* Delays in communication
* Staff overload

CampusBot automates all of this, saving time and improving efficiency.

---

## 🛠️ How It Works

1. School sets up CampusBot
2. Parents send messages on WhatsApp
3. Bot replies instantly with accurate information
